# Simple Interest Calculator

A simple shell script to calculate the simple interest based on the principal amount, the annual rate of interest, and the time period in years.

## Description
This project takes three inputs from the user:
1. **Principal Amount** (P)
2. **Annual Rate of Interest** (R)
3. **Time Period in Years** (T)

It then calculates the Simple Interest using the formula:
$$SI = \frac{P \times R \times T}{100}$$
